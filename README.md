# Como funciona?
~~~~
Este projeto tem como intuito criar um jogo que gera um número aleatório;
O objetivo do jogador é acertar este número;
A cada tentativa, o contador de tentativas aumenta;
Ao final do jogo, é indicado quantas tentativas o jogador levou para acertar o número secreto.
~~~~
